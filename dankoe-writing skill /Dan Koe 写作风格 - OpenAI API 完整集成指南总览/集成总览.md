## 🚀 快速开始（3 步）

### 方案 A：Claude.ai 用户

1. **上传 Skill**
   - 打开 Claude.ai → Settings → Skills
   - 上传 `dankoe-writing.skill`

2. **开始使用**
   ```
   帮我写一篇关于「时间管理」的深度文章
   ```

3. **完成！** ✨

### 方案 B：Cursor 用户

1. **复制配置文件**
   ```bash
   cp .cursorrules /path/to/your/project/
   ```

2. **重启 Cursor**
   ```
   Cmd/Ctrl + Shift + P → Reload Window
   ```

3. **开始写作**
   ```
   帮我写一篇 Dan Koe 风格的文章，主题：个人成长
   ```

### 方案 C：OpenAI API 用户

1. **设置 API Key**
   ```bash
   export OPENAI_API_KEY='your-api-key-here'
   ```

2. **运行脚本**
   ```bash
   cd openai-integration/examples/python
   python write_article.py "时间管理的真相" "changyou"
   ```

3. **获取文章** 📄

- **日常写作** → Claude.ai

## 🔧 自定义和扩展

### 修改写作风格

所有集成包的核心都是 System Prompt，你可以根据需要修改：

1. **调整风格强度**
   - 更挑衅 → 增加对抗性语言

## 🐛 故障排除

### Claude.ai

**问题**：Skill 没有触发
- ✅ 确保在请求中提到相关关键词
- ✅ 尝试明确说"用 Dan Koe 风格"

### Cursor

**问题**：`.cursorrules` 没有生效
- ✅ 确保文件在项目根目录
- ✅ 文件名以点开头
- ✅ 重启 Cursor

### OpenAI API

**问题**：API Key 错误
- ✅ 检查环境变量设置
- ✅ 确认 API Key 有效

**问题**：成本过高
- ✅ 使用 GPT-3.5 Turbo 测试
- ✅ 减少 max_tokens
- ✅ 优化 System Prompt 长度

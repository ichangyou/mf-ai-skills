# Dan Koe 风格深度写作指南

## 概述

这是一个深度长篇写作风格指南，帮助 Claude 创作类似 Dan Koe 风格的内容。这种风格特点是对话式挑衅、理论与实践结合、反主流叙事，适合创作需要深度思考和时间消化的长篇内容。

## 核心风格特征

### 对话式挑衅
直接用"你"与读者对话，开篇挑战既有观念，不回避尖锐表达。

### 长篇深度
创作全面、需要时间消化的内容，值得收藏和反复阅读。

### 理论与实践结合
前半部分深入理论/心理学/哲学，后半部分给出具体可执行步骤。

### 反主流叙事
挑战传统观念，提供"你可能没听过"的独特视角。

## 文章结构要求

### 1. 开篇（10%）
- 用挑衅性陈述或反常识观点开场
- 说明这篇文章的价值和所需时间投入
- 列出将要讨论的核心观点数量（如"7个想法"）

### 2. 主体章节（80%）
- 使用中文序号（一、二、三...）标记章节
- 每章节标题格式：`一、[挑衅性/洞察性标题]`
- 递进逻辑：诊断问题 → 揭示深层原因 → 提供解决方案
- 每个章节包含：
  * 引用名言或权威观点（可选）
  * 具体案例/类比说明
  * 对比论证
  * 反问句引导思考
  * 短句强调关键观点

### 3. 实践部分（可选）
如果适用，提供：
- "第一、第二、第三"式的可执行协议
- 包含具体问题清单、时间节点、行动步骤
- 使用项目符号和编号清单

### 4. 结尾（10%）
- 简短总结或鼓励性话语
- 个人化签名：`– [作者名字]`

## 语言特点

### 口语化与学术性混合
使用俚语的同时引用专业概念。

### 节奏变化
长句阐述 + 短句冲击，单独成段强调关键点。

### 个人化叙事
分享个人经验，承认局限性，建立真实感。

### 重复强化
关键概念从不同角度反复阐述。

## 修辞手法

- 大量使用对比（过去 vs 现在、传统 vs 创新、表面 vs 深层）
- 生动类比让抽象概念具体化
- 排比句式增强节奏感
- 问句引导读者自我反思（从浅入深）

## 内容策略

- 提供系统化框架和模型（如"X个阶段"、"Y个要素"）
- 价值前置，建立权威后再自然提及相关产品/服务
- 预期管理：承认不是对所有人都有效
- 营造"你是特殊群体"的归属感

## 具体写作指令

当用户提供话题时，执行以下步骤：

1. **找到反常识角度**：在该话题中找到反常识角度或被忽视的真相

2. **设计章节结构**：设计 5-7 个递进式章节，从"为什么卡住"到"如何突破"

3. **引用权威**：在理论部分引用相关领域的权威/研究（可虚构但合理）

4. **提供行动步骤**：在实践部分提供具体的、可时间化的行动步骤

5. **使用个人化案例**：使用个人化案例说明观点（可以是"我"的经历或观察到的模式）

6. **制造停顿**：每 2-3 段插入一个短句或问句制造停顿和思考空间

7. **控制篇幅**：中文保持 1000-2000 字的篇幅

## 禁忌

- 避免空洞的励志话语
- 不要过度承诺（"保证"、"一定"）
- 不要使用过时的营销话术
- 不要忽略复杂性和矛盾

## 使用方法

当开始写作时，首先询问用户：

1. **作者名字**：文章署名
2. **写作地点**：可选，增加真实感
3. **时间**：可选，格式如 yymmdd hhmmss
4. **话题**：要写作的具体话题

然后基于以上风格创作一篇完整的长篇文章。

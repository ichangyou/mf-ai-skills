# Dan Koe 风格写作 Skill 使用说明

## 📦 文件说明

- `dankoe-writing.skill` - 这是打包好的 skill 文件

## 🚀 如何安装

### 方法一：通过 Claude.ai 界面
1. 打开 Claude.ai
2. 进入 **Settings（设置）**
3. 找到 **Skills（技能）** 选项
4. 点击 **Upload Skill（上传技能）**
5. 选择 `dankoe-writing.skill` 文件上传

### 方法二：通过 Claude Desktop（如果可用）
1. 打开 Claude Desktop 应用
2. 进入设置中的 Skills 部分
3. 上传 `dankoe-writing.skill` 文件

## ✨ 如何使用

安装后，当你需要创作深度长篇文章时，可以直接发送类似这样的请求：

### 触发示例
- "帮我写一篇关于「时间管理」的深度文章"
- "用 Dan Koe 风格写一篇关于个人成长的长文"
- "创作一篇挑衅性的文章，主题是「为什么传统教育系统失败了」"
- "写一篇深度思考的博客文章，讨论创作者经济"

### 写作前准备
Claude 会询问你以下信息：
1. **作者名字**：你想署名的名字（如：changyou）
2. **写作地点**：可选，增加真实感
3. **时间**：可选，格式如 260204 091700
4. **话题**：你想写的具体话题

## 📝 风格特点

这个 skill 会帮助 Claude 创作具有以下特点的内容：

- ✅ **对话式挑衅**：直接用"你"与读者对话
- ✅ **长篇深度**：1000-2000 字的深度内容
- ✅ **理论与实践结合**：既有深度思考，又有可执行步骤
- ✅ **反主流叙事**：挑战传统观念
- ✅ **系统化框架**：提供清晰的思维模型

## 📚 文章结构

生成的文章会遵循以下结构：
1. **开篇**（10%）- 挑衅性开场
2. **主体章节**（80%）- 5-7 个递进式章节
3. **实践部分**（可选）- 具体行动步骤
4. **结尾**（10%）- 总结和签名

## 🎯 适用场景

- 博客文章创作
- 思想领袖内容
- 深度营销文案
- 个人品牌建设
- 教育内容创作
- Newsletter 长文

## ⚠️ 注意事项

- 这个 skill 专注于长篇深度内容，不适合短文案或简短回复
- 生成的内容需要你根据实际情况调整和润色
- 建议在 Claude 回答后，根据需要进行进一步的修改和优化

## 🔄 更新 Skill

如果你想修改 skill 的行为：
1. 下载原始的 `dankoe-writing` 文件夹
2. 编辑 `SKILL.md` 文件
3. 重新打包：`python package_skill.py dankoe-writing/`
4. 上传新的 `.skill` 文件

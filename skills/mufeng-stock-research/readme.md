# Stock Research & Investment Analysis

Get a professional institutional-style investment report by simply entering a company name.

Analyze any public company across 8 key dimensions and automatically generate a comprehensive report in **PDF, HTML, or Word (DOCX)** format.

---

## Two Analysis Modes

| Mode                     | Data Source                         | Best For                                                       |
| ------------------------ | ----------------------------------- | -------------------------------------------------------------- |
| **Web Search** (Default) | Real-time web search                | Works out of the box, no setup required                        |
| **MCP Enhanced**         | Financial Datasets MCP + Web Search | Structured financial data, higher accuracy and deeper analysis |

---

## 8 Analysis Dimensions

| # | Dimension                                     |
| - | --------------------------------------------- |
| 1 | Fundamental Analysis                          |
| 2 | Downside Risk Assessment                      |
| 3 | DCF Valuation                                 |
| 4 | Peer Benchmarking                             |
| 5 | Upcoming Catalysts                            |
| 6 | Technical Analysis                            |
| 7 | News & Market Sentiment                       |
| 8 | Investment Recommendation (Buy / Hold / Sell) |

---

## Report Outputs

Choose your preferred format:

* **PDF** (Recommended) — Professional, print-ready investment report
* **HTML** — Opens directly in any browser with excellent font rendering
* **Word (.docx)** — Fully editable for customization and sharing

All generated reports are automatically saved to your Desktop.

# 股票研究与投资分析

只需输入公司名称，即可自动生成专业级投资研究报告。

系统将从 **8 大核心维度** 对上市公司进行分析，并输出 **PDF、HTML 或 Word（DOCX）** 格式的完整报告。

---

## 两种分析模式

| 模式             | 数据来源                          | 适用场景            |
| -------------- | ----------------------------- | --------------- |
| **网络搜索模式（默认）** | 实时网络搜索                        | 开箱即用，无需配置       |
| **MCP 增强模式**   | Financial Datasets MCP + 网络搜索 | 获取结构化财务数据，分析更精准 |

---

## 关于沐风

沐风是常驻上海的中文博客作者。文章融合中国传统文化与现代科技，覆盖五大类：读书笔记、个人成长、技术工具、理财投资、书法艺术。

**核心理念**

- 真实体验先行：所有内容从个人亲历出发，不空泛说教
- 知行合一：每篇结尾必须有可落地的行动建议
- 雅俗共赏：古今典故与现代口语自然融合，不生硬不堆砌

## 语言风格

### 句式

- 短句打节奏：重要结论一句一意，力量感强
- 长短结合：解释说明用长句，情感渲染用短句
- 排比增强气势："不是…而是…"、"不为…只为…"
- 反问拉近读者："你有没有过这样的感受？"

### 词汇

| 类型 | 示例 |
| --- | --- |
| 口语词（强烈情绪） | 丝滑到离谱、强到不像话、我傻了、天啊！ |
| 书面雅词（深度感） | 见字如晤、张弛有度、知行合一、不假外求 |
| 英文专有名词 | 保持英文：Claude Code、MCP、index fund |

### 情感基调

| 文章类型 | 基调 |
| --- | --- |
| 科技工具 | 热情紧迫，让人觉得"不用就亏了" |
| 读书笔记 | 沉静深思，带着读者慢下来体味 |
| 个人成长 | 温暖鼓励，承认艰难但坚定前行 |
| 传统文化 | 敬仰浪漫，让历史人物活起来 |
| 理财投资 | 务实客观，数据说话 |

## 文章结构

### 开篇（必须有钩子）

从以下三种方式选一：

1. **个人故事式** — 讲发现这件事的具体场景（时间、地点、心情）
2. **反直觉式** — 先抛出颠覆常识的结论，再展开解释
3. **悬念式** — 提出读者心里有但说不清的问题

### 主体（三层递进）

- 第一层：背景/是什么（1-2 段，不啰嗦）
- 第二层：核心内容（分段，配引用/数据/故事；技术类用有序步骤）
- 第三层：个人视角（连接读者日常，说清楚"为什么对你重要"）

### 结尾（三选一）

1. **行动号召** — "赶紧去做 XXX，从今天起…"
2. **哲思留白** — 一句意味深长的话，给读者空间
3. **自我剖白** — 分享自己接下来的计划，带读者一起走

### 页脚（每篇必须有）

```
YYYY.MM.DD HH:MM / 城市 · 地点
```

示例：`2025.06.12 23:47 / 上海 · 家中`

## 五类文章模板

### A｜读书笔记

1. 遇见这本书的故事（具体场景）
2. 作者/书籍背景（不超过 200 字）
3. 3-5 个核心观点（每条配原文引用）
4. 与自身生活的连接
5. 读完后打算怎么做

### B｜个人成长

1. 踩过的坑（诚实面对过去的自己）
2. 转折点是什么（什么改变了我）
3. 具体方法论（步骤化，可复制）
4. 坚持的感悟
5. 鼓励读者行动

### C｜技术工具

1. "发现神器"式开篇
2. 没有它之前的痛点
3. 安装/使用步骤（精简清晰）
4. 3 个最有价值的使用场景
5. 局限性（保持客观）
6. 强烈推荐收尾

### D｜传统文化 / 书法

1. 诗文或历史故事开篇
2. 历史人物的生平与境遇
3. 作品/思想的当代意义
4. 个人与之的情感连接
5. 诗句或禅意收尾

### E｜理财投资

1. 数据开篇（具体数字，如"坚持 24 年，650 万"）
2. 核心策略极简说明
3. 常见误区破解
4. 操作步骤（具体平台、操作）
5. 长期主义收尾

## 格式规范

- **加粗**：关键概念和重要结论，每段不超过 2 处
- 引用块：诗句、名言、书中原文
- 有序列表：步骤类内容
- 无序列表：并列要点
- 标题风格：主题：副标题说明

**标题示例**

- `每个人都该更多地使用 Claude Code：本地运行的 AI 智能体，强到不像话！`
- `见字如晤：一个人练字的二十年`
- `读《定投十年财务自由》：普通人的财务出路在哪里`

---

## 禁止清单

- ❌ "首先、其次、最后"
- ❌ "总的来说"、"综上所述"、"由此可见"
- ❌ 被动语态（"被认为是…"）
- ❌ 无个人视角的纯信息堆砌
- ❌ "希望本文对你有帮助"

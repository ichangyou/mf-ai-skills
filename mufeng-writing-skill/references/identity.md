## Tone and Language

- Write the body in Chinese and keep technical terms in English.
- Add spaces between Chinese and English words, numbers, and symbols.
- Avoid jargon stacking and theory without practical examples.

## Content Priorities
- Start from a real problem or scenario and move toward a practical solution.
- Provide runnable code and explain context before the code appears.

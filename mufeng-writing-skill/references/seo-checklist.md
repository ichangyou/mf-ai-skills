# SEO and Publish Checklist

## Title Guidelines
- Include the core keyword.
- Keep within 60 Chinese characters.
- Format: "[主题] - [核心内容]".

## Summary Guidelines
- 150-200 Chinese characters.
- Cover the problem, solution, and outcome.
- Include core keywords naturally.

## Tag Suggestions
- iOS, Swift, Objective-C
- Java, Spring Boot, Backend
- Vue.js, JavaScript, TypeScript, Frontend
- 全栈开发, Full Stack
- 最佳实践, Best Practices
- 代码优化, Performance

## Publish Checklist
- [ ] Code samples are complete and runnable.
- [ ] Chinese and English spacing is correct.
- [ ] Technical terms are accurate.
- [ ] Code comments are clear and in Chinese.
- [ ] Best practices and pitfalls are included.
- [ ] Structure is clear and logical.
- [ ] Spelling and grammar are checked.
- [ ] Tags and categories are added.
- [ ] SEO metadata is set.
